Welcome to the documentation of ``mesh_illustris``!
=======================================================

``mesh_illustris`` is a toolkit for analyzing `Illustris <https://www.illustris-project.org/>`_ (and also `IllustrisTNG <https://www.tng-project.org/>`_) data with mesh. The goal of ``mesh_illustris`` is to **quickly** load a subset (e.g., a box or sphere) of particles/cells with **minimal** amount of memory. :doc:`Get started <get_started>` now!
